**Placement:** Put all files in `public/blog/`. Blog markdown references them as `![alt text](/blog/filename.jpg)`.

**Sizing:** Aim for at least 1200px wide for hero images; 800px wide is fine for in-body images. Compress for web (e.g. 80–85% quality) to keep page load fast.
